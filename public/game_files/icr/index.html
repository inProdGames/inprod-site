Inverse Cube Runner

Inverted Productions > Productions > Inverse Cube Runner


Zachary Yaro had the idea for Inverse Cube Runner when he was playing Cube Runner and thought it would be nice if it could be a multiplayer game. Rather than looking for one, he made one.

You must have the latest version of Java installed to play this game.
Release date: 2010-02-15
Current version: 5.1


Created and programmed by Zachary Yaro
