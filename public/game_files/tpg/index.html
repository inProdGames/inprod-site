The Two-Player Game

Inverted Productions > Productions > The Two-Player Game


The Two-Player Game is a set of minigames designed for two players. You can choose from categories like survival, player-vs.-player, and point games. You and another player then compete in several games until a winner is decided. Games will be added/updated over time.


Current version: Low effort demo
Status: work in progress
